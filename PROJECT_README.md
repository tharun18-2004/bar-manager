# BAR LOGIC – Bar Management System

A full-stack bar management platform built to manage bar operations efficiently.

## Features
• POS Billing System  
• Inventory Management  
• Stock Register  
• Expense Tracking  
• Sales Analytics Dashboard  
• Invoice PDF Generation  
• Owner Analytics  

## Tech Stack
• Next.js  
• React  
• TypeScript  
• Supabase  
• PostgreSQL  
• Tailwind CSS
